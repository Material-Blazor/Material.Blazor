IMBToastService

Summary

Manages toast notification.

Details

  • See the toast article for usage details;
  • Requires an MBAnchor component and will throw an exception when you attempt to show a toast notification if this isn't found; and
  • Accepts an optional MBToastServiceConfiguration.
