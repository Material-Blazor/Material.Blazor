OnValueSet

Derived components can use this to get a callback from the Value setter when the consumer changes the value. This allows a component to take action with Material Theme js to update the DOM to reflect the data change visually. An example is a select where the relevant list item needs to be automatically clicked to get Material Theme to update the value shown in the <input> HTML tag.
