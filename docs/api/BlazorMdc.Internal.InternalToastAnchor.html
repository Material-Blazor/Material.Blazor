Class InternalToastAnchor

An anchor component that displays toast notification that you display via ShowToast(MTToastLevel, String, String, Nullable<MTToastCloseMethod>, String, String, IMTIconFoundry, Nullable<Boolean>, Nullable<UInt32>). Place this component at the top of either App.razor or MainLayout.razor.
