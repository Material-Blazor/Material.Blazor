Enum MTIconFAStyle

Sets the Font Awesome style.

See FAFoundry(Nullable<MTIconFAStyle>, Nullable<MTIconFARelativeSize>), BlazorMdc.Internal.IconFA and BlazorMdc.IconFoundryFA

MTCascadingDefaults has a default of Solid (all other styles except Brands require a paid-for Font Awesome PRO licence)
