Class MBToastAnchor

An anchor component that displays toast notification that you display via ShowToast(MBToastLevel, String, String, Nullable<MBToastCloseMethod>, String, String, IMBIconFoundry, Nullable<Boolean>, Nullable<UInt32>, Boolean). Place this component at the top of either App.razor or MainLayout.razor.
