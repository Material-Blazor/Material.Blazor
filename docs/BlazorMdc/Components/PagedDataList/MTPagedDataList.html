MTPagedDataList<TItem>

Summary

A templated component for paging generic data lists using MTPaginator. Uses the "Wig Pig" nested coding pattern and can either render list items one by one or wrap a component such as MTList or MTDataTable.

Details

  • Paginates an IEnumerable of a data item that you define (TItem);
  • Can either apply basic formatting of that data on your behalf, or allow your render fragment access to the skip/take paginated section of your dataset for you to render;
  • Uses MTSlidingContent to apply subtle left/right, fade out/in animation to indicate page transitions to the user;
  • Integrates with MTPaginator and its parameterization, including two-way binding for the page number;
