Interface IMTToastService

Interface for the BlazorMdc toast service, developed from the code base of Blazored Toast by Chris Sainty. Works in conjunction with a MTToastAnchor that must be placed in either App.razor or MainLayout.razor to avoid an exception being thrown when you first attempt to show a toast notification.

Throws a InvalidOperationException if ShowToast(MTToastLevel, String, String, Nullable<MTToastCloseMethod>, String, String, IMTIconFoundry, Nullable<Boolean>, Nullable<UInt32>) is called without an MTToastAnchor component used in the app.

You can optionally add configuration when you add this to the service collection:

services.AddMTToastService(new MTToastServiceConfiguration()
{
    Postion = MTToastPosition.TopRight,
    CloseMethod = MTToastCloseMethod.Timeout,
    ... etc
});
