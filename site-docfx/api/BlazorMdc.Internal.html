ClassAndStyleMapper

The definition of ClassAndStyleMapper and ClassAndStyleMapperExtensions ultimately define a list of parameterless functions that conditionally return a string if true, a null if false.

The Func<string> items are enumerated and evaluated when either ToClassString() or ToStyleString() are called.
