BlazorMdc (pre-release)

Lightweight Material Theme razor components for Blazor

BlazorMdc is a lightweight Material Theme web development platform (version 7.0.0) component library for ASP.NET Blazor that is rigorously faithful to the Material Theme's design philosophy, markup and code usage. Material Theme has very specific and detailed guidance showing web designers how to build web apps with HTML, CSS and JavaScript. Since neither Blazor for WebAssembly are directly supported, Blazor developers either need to roll their own components or use a component library such as BlazorMdc.

  • Visit our website at https://blazormdc.com;
  • View the project API documentation at https://docs.blazormdc.com; and
  • Blazor MDC is available at https://www.nuget.org/packages/BlazorMdc.
  • If you intend to submit pull requests please note that we use Git Flow with develop and main branches. You can read detailed development environment instructions on our docs site.
